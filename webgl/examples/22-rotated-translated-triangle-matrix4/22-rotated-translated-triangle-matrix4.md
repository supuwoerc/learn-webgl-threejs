## 方法说明
* 注意，矩阵乘法运算不满足交换律，但是满足结合律
* 变换后坐标 = 变换矩阵 * 原始坐标

> 先平移再旋转矩阵处理过程为
平移后坐标 = 平移变换矩阵 * 原始坐标，
> 再做旋转，
> 平移&旋转后坐标 = 旋转变换矩阵 * (平移变换矩阵 * 原始坐标)
> 因为满足矩阵乘法满足结合律
> 平移&旋转后坐标 = (旋转变换矩阵 * 平移变换矩阵) * 原始坐标

```javascript
    // 先平移再旋转（注意矩阵操作顺序）
    matrix.setRotate(60, 0, 0, 1)
    matrix.translate(0.2, 0.2, 0)
```
```javascript
    // 先旋转再平移（注意矩阵操作顺序）
    matrix.setTranslate(0.2, 0.2, 0)
    matrix.rotate(60, 0, 0, 1)
```
